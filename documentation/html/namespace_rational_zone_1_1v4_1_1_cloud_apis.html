Detailed Description

RationalZone.v4.CloudApis is a general purpose wrapper library for several OAuth based cloud API service. Purpose is not be 100% wrapper on any but rather provide consistent horizontal abstraction that allows you to utilize multiple services transparently. The library basically handles the differences in OAuth implementation like the parameter names and security practices.
